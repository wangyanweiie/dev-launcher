# 启动脚本

按平台选用对应目录中的脚本（工作目录均为仓库根目录）。

| 平台 | 目录 | 双击 | 命令行 |
|------|------|------|--------|
| macOS | [macos/](macos/) | `启动 Dev Launcher.command` | `bash scripts/macos/start-dev-launcher.sh` |
| Windows | [windows/](windows/) | `启动 Dev Launcher.bat` | `scripts\windows\start-dev-launcher.bat` |

Linux 可使用 `scripts/macos/start-dev-launcher.sh`（与 macOS 相同）。
